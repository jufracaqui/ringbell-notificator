# ringbell-notificator

Receive Telegram notifications when someone rings you ringbell

---

Follow https://github.com/witnessmenow/Universal-Arduino-Telegram-Bot steps to setup the IDE and create a bot

Use a [ZMCT103C](https://es.aliexpress.com/item/32914725175.html) sensor to read current on your ringbell cables

---
By default, connect the output pin of the sensor to pin D34 on the ESP32

[Voltage Divider](https://imgur.com/EIC3Hgm) for reading 3.3V from 5V output on the sensor
